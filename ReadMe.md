#如何编译？
编译本源码需要MinGW和Python。
在本目录执行：
python waf configure build install
生成的东东会保存在release目录。

#如何运行？
程序运行还需要额外的数据文件，你可以到：
http://pan.baidu.com/s/1mgkOinQ
下载。
将数据解压到release目录，即可运行程序。
